# 📋 TaskMaster Pro

Welcome to the **TaskMaster Pro** repository! 

This is a personal learning project aimed at building a robust, professional-grade desktop task management application. The project is designed to master the .NET Ecosystem, specifically focusing on building modern graphical interfaces, ensuring local data persistence, and strictly separating business logic from the UI.

## 🚀 Technologies Used
* **Framework:** .NET 8.0 / C# 12.0
* **Frontend:** WPF (Windows Presentation Foundation) with XAML
* **Database:** SQLite
* **Data Access:** Entity Framework Core (EF Core)
* **Architecture:** MVVM (Model-View-ViewModel)
* **Testing:** xUnit & Moq
